# I-E7.1
I&E challenge Market analysis using IT tools - team 7.1

# Setup

Create a virtualenv.

```bash
$ python3 -m venv venv
```

Activate it.

```bash
$ source venv/bin/activate
```

Install the required dependencies.

```bash
$ pip install -r requirements.txt
```

Start Jupyter Notebook
```bash
$ jupyter notebook
```

Happy Coding.
